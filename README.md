# Paradise Nursery 🌿

A React-based e-commerce shopping cart application for a houseplant nursery.

## Project Overview

**Paradise Nursery** is a full-featured plant shopping application built with React and Redux Toolkit. Browse beautiful houseplants, add them to your cart, and manage your purchases with ease.

## Features

- 🌱 Browse 18+ unique houseplants across 3 categories
- 🛒 Redux-powered shopping cart
- ➕ Add/remove items and adjust quantities
- 💰 Real-time cart total calculation
- 📱 Responsive design

## Tech Stack

- React 18
- Redux Toolkit
- React Router DOM
- CSS3

## Getting Started

```bash
npm install
npm run dev
```
